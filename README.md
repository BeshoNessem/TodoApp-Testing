# Todo App Website 📝

## 📋 Overview
A simple web app for managing tasks, allowing users to add, edit, delete, and mark tasks as complete.

## 🚀 Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
